# 株価シミュレーション＆AI仮想投資アプリ

**市場を動かす戦略を、あなたの手に。**

---

## 💡 ビジョン

現実のお金を使わず、AIの投資戦略を安全に試せる。
株価の動きを再現し、AIが週ごとに戦略を進化させる。
あなたはシグナルに従うだけ。

---

## 🚀 核心機能

* **株価を再現**：統計手法でリアルな価格変動をシミュレーション
* **AI戦略更新**：最新の市場動向をもとに毎週戦略を再構築
* **結果が届く**：仮想投資の成果を週1レポートで自動配信

---

## 🎯 なぜ作ったのか

投資戦略を「安全に」「すぐに」「誰でも」試せる場所が必要だったから。
